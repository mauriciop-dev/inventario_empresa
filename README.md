# 📦 Sistema de Inventario en Consola (ProDig - Prospectiva Digital)

Este sistema de inventario permite agregar, buscar, restar y visualizar productos mediante un menú interactivo en consola.

---

## 🛒 ¿Qué hace este sistema?

- Agrega productos con su cantidad.
- Permite restar unidades de un producto.
- Busca productos por nombre.
- Muestra el inventario completo.
- Controla errores si el producto no existe o si se intenta restar más de lo disponible.

---

## 🧠 Aprendizajes aplicados

- Diccionarios en Python (`dict`)
- Bucle `while` con menú interactivo
- Entrada de datos con `input()`
- Validaciones con `if`, `else`
- Práctica de lógica real para automatización
